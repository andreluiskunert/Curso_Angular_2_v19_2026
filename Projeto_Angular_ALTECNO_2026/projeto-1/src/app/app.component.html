Pipes em Angular

Pipes no Angular

Nesta aula, foi apresentado um dos recursos mais importantes do Angular: os Pipes. Eles são utilizados para transformar e formatar dados antes de serem exibidos na interface do usuário.

O que são Pipes?

Pipes permitem manipular dados apenas na visualização, sem alterar a fonte original. Eles são usados dentro do template do Angular, principalmente em expressões de interpolação.

Como os Pipes funcionam?

Para utilizar um pipe, usamos o símbolo | (pipe) dentro do template. Com isso, é possível aplicar diferentes transformações nos dados exibidos.

Exemplos de Pipes Padrão

UpperCase e LowerCase

O pipe uppercase transforma textos em letras maiúsculas, enquanto o lowercase transforma em letras minúsculas.

Json Pipe

O pipe json permite visualizar objetos completos diretamente na tela, facilitando a análise dos dados vindos da fonte.

Currency Pipe

O pipe currency é utilizado para formatar valores monetários. No Angular mais recente, basta informar a moeda (como BRL) para que o valor seja exibido corretamente.

Mesmo com a formatação aplicada, os dados originais permanecem inalterados na fonte.

Pipes Assíncronos

O pipe async é usado para exibir dados que ainda estão sendo carregados, geralmente vindos de serviços assíncronos.

Conclusão

Os pipes facilitam a exibição de dados no Angular, tornando a interface mais clara e amigável para o usuário. Na próxima aula, será abordada a criação de pipes personalizados.
