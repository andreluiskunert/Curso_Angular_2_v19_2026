Anatomia de um Projeto Angular

1. Criação do Projeto Angular

A aula mostra como criar um projeto Angular desde o início usando o terminal. Primeiro, escolhe-se uma pasta para armazenar os projetos e depois utiliza-se o comando ng new nome-do-projeto.

Exemplo: ng new projeto1
